# cloud-support-lab
Projeto prático simulando o dia a dia de um Analista de Suporte Cloud Jr, com foco em troubleshooting, Linux e redes.

##  Objetivo
Simular incidentes reais em um ambiente Linux e aplicar resolução de problemas.

## Tecnologias utilizadas
- Linux
- Terminal 
- Redes básicas
- Controle de permissões

## Cenários simulados
- Falha de permissão de arquivos
- Serviço fora do ar
- Problemas de acesso

## Habilidades demonstradas
- Diagnóstico de problemas
- Investigação de incidentes
- Resolução de falhas
- Uso de comandos Linux

## Estrutura do projeto
- incidentes/: documentação dos problemas e soluções
- scripts/: comandos utilizados

## Aprendizados
- Importância de validar solução
- Análise de permissões
- Troubleshooting em ambiente Linux
